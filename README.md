# iOS-Movie-App

A movie app that shows popular movies and their informations. You can add movies to your favorites.<br/>
 The Api that has been used : https://www.themoviedb.org/ . <br/>
Tech stack that has been used;

- Swift
- UIKit
- MVVM
- RxSwift
- URLSession
- Core Data
- Cocoapods
- Kingfisher
